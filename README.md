# Momo Store
Контейнеризированное приложение интернет-магазина, состоящее из:
Backend – Go API на порту 8081
Frontend – Vue.js SPA, обслуживается Nginx на порту 80

# Требования
    Docker 20.10+
    Docker Compose 2.0+

# Быстрый запуск

1. Клонируйте репозиторий:
    git clone <repo-url>
    cd momo-store

2. Создайте файл .env с секретами (пример):
    JWT_SECRET=ваш_ключ

3. Запустите приложение:
    docker-compose up -d
    Доступные профили: dev, prod.

4. Откройте в браузере: http://localhost

# Для обновления до последних версий базовых образов пересоберите с флагом --pull:
docker-compose build --pull

# Запуск с масштабированием. Балансировщик nginx автоматически распределит нагрузку между репликами благодаря Docker DNS и конфигурации upstream.
docker-compose --scale backend=3 --scale frontend=2


Компонент	Базовый образ	    Размер	    Описание
Backend	    alpine:3.23.4	    15-20 MB	Скомпилированный Go-бинарник, без лишних зависимостей
Frontend	nginx:1.25-alpine	25-30 MB	Только статика и минимальный веб-сервер

Multi-stage сборки гарантируют, что в финальный образ попадают только артефакты, необходимые для работы.
В CI/CD пайплайне (GitHub Actions) используется Trivy для проверки образов на критические и высокие уязвимости перед деплоем.
